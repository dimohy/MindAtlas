# Avalonia

> 크로스플랫폼 .NET UI 프레임워크, XAML 기반 선언적 UI 지원

Tags: #Avalonia, #dotnet, #XAML, #Cross-platform

Avalonia는 크로스플랫폼 [[.NET]] UI 프레임워크로, [[XAML]] 기반의 선언적 UI를 지원합니다. Windows, macOS, Linux뿐만 아니라 iOS와 Android에서도 동작합니다. 주요 특징으로는 고급 [[스타일링 시스템]], [[데이터 바인딩]], 그리고 [[컨트롤 템플릿]]을 제공합니다. 데스크톱과 모바일에서 일관된 UI를 만들고자 할 때 적합합니다.
