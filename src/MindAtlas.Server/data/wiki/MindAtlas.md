# MindAtlas

> 개인용 LLM 기반 위키(백과사전) 콘셉트의 지식관리 시스템

Tags: #LLMWiki, #개인백과사전, #지식관리, #MindAtlas

MindAtlas는 LLM Wiki를 콘셉트로 한 개인 백과사전이다. 사용자는 개인 지식과 노트를 구조화하여 저장하고, 대화형 LLM을 통해 검색·요약·연결할 수 있다. 핵심 기능으로는 항목 간 [[wikilinks]] 자동 제안, 자연어 질의 응답, 그리고 개인화된 지식 그래프 생성이 있다. 확장 가능성으로는 UI 프레임워크(예: [[Avalonia]]) 연동이나 인증/테스트 관련 모듈(참고: [[Token auth test]]) 통합이 있다.
